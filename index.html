Rlinux

As a sub-corporation of RTA Incorporations, Rlinux is a coporation that strives to have a better understanding of the common file system. Rlinux is derived on Unix-based operating systems which can support the libraries to manage the file system. What is more, kids can use the file system for testing and for learning basic security skills, as well as take a leaf out of my book and use this as a base to design their own projects. Click the R logo to download the latest version of Rlinux.
